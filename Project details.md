# Medicine Reminder & Health Tracker

## Project Description
Medicine Reminder & Health Tracker is a healthcare application that helps users manage their medications and health records. The system provides medicine reminders, dosage tracking, health monitoring, and appointment management to improve patient care and medication adherence.

## Features
- Medicine Reminder
- Medication Tracking
- Health Record Management
- Appointment Scheduling
- Notification Alerts
- User Authentication

## Technologies Used
### Front End

- JavaScript

### Back End
